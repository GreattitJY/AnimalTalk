## 2. 커밋 컨벤션

| 키워드   | 사용 시점                              |
| :------- | :------------------------------------- |
| Feat     | 새로운 기능 추가                       |
| Fix      | 버그 수정                              |
| Docs     | 문서 수정                              |
| Style    | 세미콜론 누락 등 기능 수정이 없는 경우  |
| Design   | 사용자 UI 디자인 변경 (CSS 등)         |
| Test     | 테스트 코드, 리팩토링 테스트 코드 추가  |
| Refactor | 코드 리팩토링                          |
| Build    | 빌드 파일 수정                         |
| Setting  | 패키지 매니저 수정, 개발 환경 설정 등   |
| Rename   | 파일 혹은 폴더명을 수정만 한 경우       |
| Remove   | 파일을 삭제한 경우                     |
